# neurodivergent

This project demonstrates a **web application designed for neurodivergent users**, focusing on accessibility and Human-Computer Interaction (HCI) principles.

It showcases features that **other apps often miss**, helping users with ADHD, dyslexia, sensory sensitivities, or other neurodivergent conditions.

---

## Features

- **Contrast Control** – Low, Medium, High contrast modes to reduce visual stress.
- **Dyslexia-Friendly Font** – Option to switch to `OpenDyslexic` font for easier reading.
- **Font Size Adjustment** – Increase or decrease text size.
- **Line-Height Adjustment** – Control spacing between lines for easier readability.
- **Word Spacing Control** – Adjust spacing between words.
- **Scroll Speed Control** – Slow down or speed up automatic scrolling.
- **Emoji Explanation Toggle** – Explains emojis to avoid misinterpretation.
- **Background Dimming** – Neutral background reduces distractions and sensory overload.
- **Current Paragraph Highlighting** – Focus one paragraph at a time while scrolling.
- **Responsive Design** – Works on mobile and desktop.

## How to Use

1. Clone or download this repository.
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge).
3. Adjust the **accessibility settings** from the panel at the top:
   - Change contrast, font, spacing, font size, line height, scroll speed, toggle emoji explanations, and dim background.
4. Enjoy a calm and accessible reading experience!

---

## Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Pastel, calm design with contrast modes
- **JavaScript** – Dynamic controls for accessibility features

---

## Purpose

This project is a **proof-of-concept** to demonstrate accessibility features for neurodivergent users. It is designed to inspire developers to **add these features to mainstream applications**.

It’s **not a production-ready app**, but an example of thoughtful HCI design.
